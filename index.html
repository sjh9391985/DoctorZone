프로젝트 요약

  • 프로젝트 이름: Doctor Zone (ver 1.0)
  • # 반응형 웹 개발
  • - MSA 아키텍처 기반의 보험 청구 시스템 개발
  • - 자바에이전트를 통한 외부 연동 및 SAM 파일생성
  • - 심사평가원 진료비 청구프로그램 연동 및 이팜모바일 QR 생성
  • 트러블 슈팅
  • #1

    - 파일, QR 생성시 한글, 특수문자, 영어 등 byte 크기를 고려해서 생성하는데 MySql과 Java에서 사용하는 바이트크기가 다른 이슈가 있었음.

  • - 각각의 인코딩 byte 크기를 확인 후 Mybatis 쿼리에서 바이트 변환하여 데이터 가져오는것으로 해결
  • #2

    - 개발기에서는 OS를 mac을 사용하고 운영측에서 고객사는 window OS를 사용하여 파일 업로드 다운로드 이슈가 있었음.

  • - 자바 에이전트를 제작하여 고객사측에서 필수 에이전트 프로그램으로 설치 후 외부 연동을 통해 이슈를 해결
  • Project URL: 출시예정

프로젝트 역할

더존비즈온 헬스케어 사업본부의 의료시스템 반응형 웹 개발이며 MSA 아키텍처 기반으로 보험심사 모듈 및 원무화면의 일부 기능 개발에 참여

Dortor Zone 보험심사 시스템의 송신/수신 화면 반응형 UI 웹 화면 개발, ajax 요청통신 공통함수 개발

보험심사 송신, 수신 화면의 SAM FILE 생성, 삭제, 추가를 위한
자바에이전트 제작을 통한 외부 연동

심사 청구 프로그램 연동을 위해서 custom uri scheme 사용과 윈도우 배치프로그램을 제작

원무 프로그램에서 사용하는 이팜 모바일 EDB QR 제작 참여
